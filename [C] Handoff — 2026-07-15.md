# Futures Financials Inc. Website — Handoff Doc
*Sessions: 2026-07-14 → 2026-07-16. This is a full current snapshot (supersedes earlier drafts of this file).*

## What this project is

A website for **Futures Financials Inc.**, Aren's student-led nonprofit focused on financial literacy education. Quick facts baked into the site copy: 18 chapters, 7 countries, 450+ volunteers, founded 2022.

The original brief: remodel the site's **structure only** (navigation style, section rhythm, card layouts, whitespace, calm nonprofit tone) after **cancerkidsfirst.org (CKF)** — never copy its wording or content. Build with placeholder copy first, swap in real content as Aren supplies it. As of now nearly all placeholder copy and imagery has been replaced with real content.

## Where it lives & stack

```

02 Projects/Futures Financials Website/Futures Financials Photos/   ← source photos (officer headshots, group photo, teaching photos, partner logos, featured-media stills)
```

**Plain HTML/CSS/JS, no build step, no framework** — Aren's explicit, standing choice over Next.js. Do not introduce build tooling, dependencies, or a framework. Runs locally via a small Python dev server (see "Running it locally").

## Key decisions made (cumulative)

1. **Stack**: Plain HTML/CSS/JS, no framework — Aren's explicit choice.
2. **Fonts**: CKF's actual typefaces, self-hosted in `fonts/` — **Open Sauce One** + **Libre Baskerville italic**. No external font requests.
3. **Palette**: green + white + black primary; **gold only as a small accent** — never a dominant fill. True kelly/grass green (hue ~132°). WCAG AA checked. Treat as settled.
4. **Compliance footer line**: every footer carries `... registered 501(c)(3) nonprofit organization (EIN: 33-1515809).`
5. **No-cache dev server**: `server.py` sends `Cache-Control: no-store`, uses assigned `PORT`/autoPort.
6. **Photo rule**: no repeated photos across the site unless a page legitimately reuses one already tracked in the Project Overview's photo map.
7. **Image processing**: real assets resized/compressed with `sips`; partner logos auto-trimmed via Pillow.
8. **Founder identity resolved**: Danny Jang is the founder ("Founder & Executive Chairman"); Hans is Co-CEO.
9. **AI-media rule (standing)**: AI imagery is always visibly stylized and labeled (chip or caption), never presented as real students/volunteers/events. Only real photos may depict real people or events.
10. **Bio pages redesigned (this session)**: replaced the full-width-portrait two-column layout with a **CKF-style brand panel** — deep-green background, concentric ripple rings (the same motif used on interior page heroes), name in large white type with a gold rule, and the portrait in a capped-width (~340px) rounded card with a bottom caption scrim (name + role in gold). Verified on both long bios (Hans) and short placeholders (Aren), and on mobile (photo card stacks on top).
11. **Guarantee-gap infographic replaced with a real US tile map (this session, major change)**: the old 50-cell abstract waffle grid was swapped for a **geographic US tile-grid cartogram**, structurally modeled on the NGPF Live U.S. Dashboard's own map (each state a labeled square placed in its real position, 12×8 grid) but rebuilt entirely in our green/white theme — no NGPF artwork or blue color used. Green fill = state guarantees a personal finance course; white outline = no guarantee yet. Every one of the 51 tiles (50 states + DC) is individually verified against the live NGPF dashboard (read directly off their map, state by state) — the count is exactly 30 green / 21 outline, matching the page's "30 of 50" stat. Each tile has a hover/keyboard-focus tooltip showing the full state name + status, plus a full aria-label on the whole grid for screen readers. Kept from the old version: the "30 of 50" number, the legend, the NGPF source citation link, and the staggered scroll-reveal animation (respects reduced-motion). Markup/CSS: `.usmap` / `.us-state` in `css/styles.css` (replaces the old `.state-cells` rules); JS hook in `main.js` still targets `#gap-cells` but now selects `.us-state` instead of bare `<i>` cells.
12. **GRA (Global Recognition Awards) badge relocated to the footer (this session)**: originally a standalone band between the homepage closing CTA and footer; moved into the **footer-brand column, under the mission line, on all 17 pages** (`.footer-award` in CSS) so it's visible sitewide rather than homepage-only. One badge image (`images/gra-badge.png`) covers both 2025 and 2026 since the artwork itself is year-agnostic — text reads "Honored 2025 & 2026."
13. **Featured Media real photos added (this session)**: the 3 real event photos Aren supplied (`Futures Financials Photos/FEATURED MEDIA *`) were processed with `sips` into `images/news-teen-teach-in.jpg`, `images/news-award-story.jpg`, `images/news-fox5.jpg` and placed in the matching "In the News" cards on **both** `index.html` and `media.html` (previously only reasoned about for Home; media.html was missed in an earlier pass and has since been fixed). These replace the old dashed placeholder boxes.
14. **FOX 5 timestamp notes (this session)**: added a small muted "Skip to 14:48" / "Skip to 7:30" note (`.link-note` class) next to each FOX 5 segment link on both Home and media.html, per Aren — the segments are much longer than the actual feature. Deliberately did **not** append unverified `#t=` URL fragments to the links themselves (no way to confirm FOX 5's player honors that syntax); the note is informational text only, not a working deep link.
15. **Program preview tiles**: the 3 homepage icon tiles were replaced with AI-illustrated concept art (`images/illo-program-*.jpg`), each labeled as illustrated/concept art — never presented as real classroom photos.
16. **Partner marquee**: pure-CSS continuously drifting logo strip on Home using all 9 existing partner logos, pauses on hover, static wrap under reduced motion. No new dependency.
17. **Curved dividers & typography elevation**: corner-sweep SVG-style dividers added at dark/light section transitions (impact, success-stories, homepage stats band); interior `<h1>`s bumped to weight 700; the existing `.hl` gold-highlight class extended to more headers.
