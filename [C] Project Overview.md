## Structure
Plain HTML/CSS/JS, no build step. Pages: index, about-us, our-story, meet-the-team, programs, impact, success-stories, volunteer, start-a-chapter, donate, our-partners. Shared styles in `css/styles.css`, behavior (nav, dropdowns, stat counters) in `js/main.js`.

Layout is structurally modeled on cancerkidsfirst.org (nav grouping, section rhythm, card patterns, calm whitespace) — structure only, never its content.

## Design rules
- Fonts (same as cancerkidsfirst.org, self-hosted in `fonts/`): **Open Sauce One** (headlines at regular weight, body, UI) + **Libre Baskerville italic** (serif callouts/quotes). Both open-source.
- Palette: green + white + black primary; **gold only as small accent** (tag pills, Donate button, icons, highlighted words).
- Pill buttons, rounded cards, alternating white / soft-green / deep-green sections.
- Image slots are dashed placeholder boxes (`.img-slot`) — swap for real assets when Aren supplies them.
